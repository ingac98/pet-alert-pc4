# PetAlert PC4

Proyecto corto para la Practica Calificada 4 del curso de Desarrollo de Software.

## Descripcion

PetAlert es un sistema web orientado al reporte de mascotas perdidas, registro de avistamientos anonimos y simulacion de alertas cercanas.

## Funcionalidades principales

- Registro de mascotas perdidas.
- Captura de coordenadas geograficas.
- Registro de avistamientos anonimos.
- Buscador multiproposito por imagen simulado.
- Interfaz para red de cuidadores.

## Patrones de comportamiento usados

- Command
- Observer
- Strategy
- Chain of Responsibility
- State

## Arquitectura

Cliente - Servidor con API REST.
